# Code of Conduct

This project follows the **Contributor Covenant**, version 2.1. The full
text is published at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be respectful, assume good intent, focus on the work, and keep
discussion on-topic. Disagreement is welcome — personal attacks are not.

## Scope

These expectations apply to all project spaces: the GitHub repository
(issues, pull requests, discussions, code review), commit messages, and
any other channel where someone is participating as a contributor or
maintainer of this project.

## Reporting

If you experience or witness behaviour that violates the Contributor
Covenant, please report it privately by opening a security advisory on
this repository:

https://github.com/buliwyf42/shellyadmin/security/advisories/new

Reports are read only by the repository maintainer. Please include enough
context to understand what happened (links, timestamps, screenshots if
relevant). Reports are taken seriously and handled discreetly.

## Enforcement

The maintainer may, at their discretion: remove or edit offending
content, lock or hide threads, and limit or revoke a participant's
ability to interact with the project. Repeated or severe violations may
result in a permanent ban from the project.
